# PVP no FPS

Многопользовательский шутер от первого лица на Unity с базовым игровым циклом: перемещение, стрельба, смерть, возрождение и подключение к игровой сессии.

Проект можно использовать как основу для создания собственного сетевого FPS.

---

## Возможности

* FPS-управление
* Сетевая игра (Host / Client)
* Стрельба и урон
* Система смерти и респавна
* Меню паузы
* Поддержка ПК и мобильных устройств
* Виртуальные джойстики для мобильного управления

---

## Управление

### Общие действия

| Действие            | ПК  | Мобильные устройства |
| ------------------- | --- | -------------------- |
| Информация о сессии | `I` | Тап тремя пальцами   |

---

### Игровая сцена

| Действие     | ПК                 | Мобильные устройства    |
| ------------ | ------------------ | ----------------------- |
| Передвижение | `W A S D`          | Левый виртуальный стик  |
| Осмотр       | Мышь               | Правый виртуальный стик |
| Прыжок       | `Space`            | Кнопка Jump             |
| Прицеливание | Правая кнопка мыши | Кнопка Aim              |
| Выстрел      | Левая кнопка мыши  | Кнопка Shoot            |
| Меню паузы   | `Esc`              | Кнопка Menu             |

---

### Дополнительные клавиши (ПК)

| Действие                 | Клавиша |
| ------------------------ | ------- |
| Уничтожить персонажа     | `K`     |
| Вернуться в главное меню | `F1`    |

---

### Главное меню (ПК)

| Действие            | Клавиша |
| ------------------- | ------- |
| Создать игру        | `1`     |
| Подключиться к игре | `2`     |
| Выход               | `F1`    |

---

## Сетевая информация

Значение ping рассчитывается на основе `EstimatedRTT` из системы Unity Netcode.
На задержку могут влиять:

* качество соединения,
* tick rate,
* время обработки транспорта сети.
